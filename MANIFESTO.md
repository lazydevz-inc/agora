## What we believe (provisional, to be ratified or replaced)

1. **AI coding fails at the input, not the output.**
   The bottleneck is not model capability. It is the gap between what a human said and what a human meant.

2. **The methods that closed this gap for 2,500 years still close it.**
   Socratic questioning, Aristotelian causation, Platonic division, Aquinian deliberation, Husserlian bracketing — these survived because they work.

3. **A tool that asks better questions produces better outputs.**
   Not because the model is smarter, but because the input is cleaner.

4. **Verification is a gate, not a hope.**
   Software is "done" when every gate passes, not when the developer says so.

5. **Bias is a service.**
   Choosing the best option for the user is *more* respectful than offering them a hundred options.
